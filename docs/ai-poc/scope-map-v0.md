# Scope Map（v0）

* 更新日：2026-03-05
* 更新者：AI
* 変更サマリ：
  * v0 初版作成（ブリーフから抽出）
  * MVP範囲を代表シナリオに準拠
  * Out of Scopeを明文化（課金/項目追加/既存アプリ統合など）

## MVP In Scope（M1までに提供）

* テナント（企業）オンボーディング（運用者による企業アカウント作成/編集/削除）
* 企業管理者のログイン（ID/パスワード）
* 企業管理者アカウント管理（作成/編集/削除）
* 従業員データCSVインポート（上書き更新、結果閲覧、エラー行提示）
* 従業員属性ラベルの作成・管理（集計の切り口）
* アンケート配信（項目固定、メール配信）
* リマインド配信（未回答者判定は内部ID）
* 従業員の回答（メールリンク、期限付き、ワンタイム、期間内再回答不可）
* 運用集計（回答状況/回答数/セグメント件数/n閾値適用）
* 結果閲覧（企業管理者向け、個票非表示、n>=5のみ表示）
* PDFレポート生成・ダウンロード（企業管理者向け）
* 問い合わせ（企業→運用者、運用者→返信）
* 監査ログ/操作ログ（配信、エクスポート、企業管理者アカウント管理、運用者の例外閲覧）

## Out of Scope（明確にやらない）

* アンケート項目の追加・編集（固定のみ）
* 既存アプリ（不適性検査）との統合・双方向連携
* 課金（請求、プラン、決済）
* オンラインサポート（チャット等のリアルタイム）
* 実名特定を目的とした追跡（個人回答の断定/公開）
* 分析ロジックの設計・開発（外部がPythonで実装しバックエンドに組込み統合）

## Later（将来やる可能性）

* SSO（SAML/OIDC）、MFA強化
* 外部連携（将来候補：既存Webアプリ等 ※統合は別途再検討）
* アンケートのテンプレ増加/分岐/多言語
* 課金・契約管理

## 依存関係（外部要因）

* メール配信基盤：SendGrid（到達率/ドメイン設定 SPF/DKIM/DMARC）
* 外部分析ロジック：Python実装（本アプリのバックエンドに組込み統合）
* インフラ：GCP（Cloud Run / Cloud SQL / Cloud Storage / Secret Manager）

## 未確定（決める期限つき）

* PDF生成方式の詳細（テンプレ/フォーマット/構成）：M0（要件確定）まで
* アンケート項目（固定の具体内容）：M0まで
* 属性ラベルの初期セット有無（完全自由か、推奨テンプレありか）：M0まで
* CSVのフォーマット仕様（必須列/任意列/キー）：M0まで
